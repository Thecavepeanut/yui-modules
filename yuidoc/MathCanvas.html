Class MathCanvas - extends Widget

Displays an arithmetical expression the way you would write it on paper.

Parameters:
config <Object> Widget configuration

Configuration Attributes

fontName - {String}

The font name to use.

fontSize - {number}

The font size to use, in em's.

func - {Y.MathFunction|String}

The function to display.

minHeight - {Integer}

The minimum height of the canvas. If the expression is taller, the height will increase to fit.

minWidth - {Integer}

The minimum width of the canvas. If the expression is wider, the width will increase to fit.
