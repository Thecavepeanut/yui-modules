Configuration Attributes

andLabel - writeonce {String}

The label for the AND button.
Default Value: "AND"

clearLabel - writeonce {String}

The label for the Clear button.
Default Value: "Clear"

combinatorMap - {Object}

A map of QueryBuilder operators to objects defining {operator,pattern}. This is needed if a variable type generates multiple values, and the values must be combined with something other than AND.
Default Value: null

fieldId - writeonce {String}

The id of the textarea form field.
Default Value: Y.guid()

fieldName - writeonce {String}

The name of the textarea form field.
Default Value: ""

formMgr - writeonce {Y.FormManager}

The FormManager to use when validating the constructed expression.
Default Value: null

insertLabel - writeonce {String}

The label for the Insert button.
Default Value: "Insert"

notLabel - writeonce {String}

The label for the NOT button.
Default Value: "NOT"

noVariableSelectedError - {String}

The error message when the user forgets to select a variable for insertion.
Default Value: 'Please choose a variable.'

orLabel - writeonce {String}

The label for the OR button.
Default Value: "OR"

parenLabel - writeonce {String}

The label for the Insert Parentheses button.
Default Value: "()"

queryBuilder - writeonce {Y.QueryBuilder}

The QueryBuilder to help the user construct the expression. The widget must not be rendered. For each variable type, the values of the configured operations must be the pattern to be inserted into the expression. {value} will be replaced by the value entered by the user.
Default Value: null

resetLabel - writeonce {String}

The label for the Reset button.
Default Value: "Cancel"

tooManyParensError - {String}

The error message for an unclosed parenthesis. context is replaced by the portion of the expression that generated the error.
Default Value: 'The expression contains an extra closing parenthesis at "{context}".'

unclosedParenError - {String}

The error message for an unclosed parenthesis.
Default Value: 'The expression contains an unclosed parenthesis.'

unmatchedSingleQuoteError - {String}

The error message for an unmatched single quote.
Default Value: 'The expression contains an unmatched single quote.'
