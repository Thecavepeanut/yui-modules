Class Plugin.Neon - extends Plugin.Base

Overrides Y.Node.show() to make it look like a flickering neon sign.

Constructor

Plugin.Neon ( config )
Parameters:
config <Object> configuration

Configuration Attributes

backgroundColor - {String}

Background (starting) color. Must be parseable by Y.Color.toRGB().

easing - {Function}

The easing to apply to the color animation.
Default Value: Y.Easing.easeIn

flickerCount - {int}

The number of flickers before the text stays visible.
Default Value: 10

textColor - {String}

Text (ending) color. Must be parseable by Y.Color.toRGB().

textShadow - {String}

Text shadow *template* for setting text-shadow CSS3 property. Use {color} to mark where color should be inserted.
